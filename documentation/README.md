# Ghost Kitchen HTML Template

A modern, production-ready HTML website template for Ghost Kitchens and Virtual Restaurant Brands.

## Features
- **Mobile First**: Optimized for seamless ordering on any device.
- **8 Pre-built Pages**: Home (2 layouts), About, Services (Menu), Blog, Contact, 404, and Coming Soon.
- **Dark/Light Mode**: User-friendly theme toggle.
- **RTL Support**: Full support for right-to-left languages.
- **Premium Design**: High-quality typography and visual layout.
- **Scroll Reveal**: Smooth entry animations for all sections.

## Folder Structure
- `assets/`: CSS, JS, Images, and Fonts.
- `pages/`: Secondary HTML pages.
- `index.html`: Primary Home page.

## Customization
- **Colors**: Edit `assets/css/style.css` :root variables.
- **Fonts**: Modify Google Fonts link in the <head> of each page.
- **Images**: Replace images in `assets/images/` or use Unsplash links in HTML.

## Performance
- Semantic HTML5 markup.
- Minimal external dependencies.
- Optimized for SEO with meta tags and structured data.

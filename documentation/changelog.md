# 📜 Changelog

All notable changes to the **Ghost Kitchen** project will be documented in this file.

## [v2.0.0] - 2026-04-07
### ✨ Added
- **Premium Documentation**: Created a full documentation suite (`README`, `Installation`, `Page Structure`, `Customization`, `Credits`, `Support`).
- **FORTIFIED v2 Architecture**: Implemented local font hosting and performance-first directory structures.
- **Enhanced SEO**: Comprehensive Meta, OpenGraph, and JSON-LD schema integration across all pages.
- **Image Optimization**: Migration to high-fidelity, optimized culinary assets.

### 🛡️ Improved
- **Navbar System**: Upgraded to `navbar-v4.js` for smoother responsiveness and theme transitions.
- **RTL Robustness**: Audited and fixed layout issues for Arabic and other right-to-left languages.
- **Accessibility**: Improved semantic HTML mapping and ARIA labels for core interactive elements.

## [v1.5.0] - 2026-04-02
### ✨ Added
- **Dark Mode**: Integrated full CSS-variable-based dark theme toggling.
- **Micro-Animations**: Added Intersection Observer-driven scroll reveal logic.
- **Service Details**: New layouts for culinary service storytelling.

## [v1.0.0] - Initial Release
- Core HTML5/CSS3 template architecture.
- Responsive grid system.
- Basic page structures (Home, About, Contact).
